# File Analyzer

Welcome to File Analyzer! Upload your files and ask questions about their content.

## Supported Files:
- CSV files
- Excel files (.xlsx) 
- Text files (.txt)
- JSON files

## Examples of questions you can ask:
- "What are the main trends in this data?"
- "Show me summary statistics"
- "What insights can you derive from this document?"
- "Explain the structure of this data"
